# Sample collection (CC0 / public domain)

These backdrops were **generated procedurally** for SlowPan (abstract gradients and
plasma, via ImageMagick) — they are not photographs or third-party art. They are
released into the public domain under
[CC0 1.0](https://creativecommons.org/publicdomain/zero/1.0/): use, modify, or delete
them freely, no attribution required.

They exist so SlowPan shows something on first run. To use your own images, add a new
folder alongside this one (e.g. `collections/my-photos/`) and set `"collection"` in
`config.json` — or leave this folder as a lightweight default.
